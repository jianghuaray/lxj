# 乐修匠社区维修管理系统 - 设计规范

> 版本：V1.0 | 更新日期：2026-05-01
> 设计风格：Organic 自然风格 + 品牌色
> 适用范围：Vue 3 + Element Plus 前端开发

---

## 一、设计系统概述

### 1.1 设计理念

采用 **Organic 自然风格**，核心特征：

- **侘寂美学（Wabi-sabi）**：不完美中的美感，柔和、温暖、自然
- **纸质纹理**：全局覆盖纸张噪点纹理，营造手工质感
- **有机形态**：不对称圆角、blob 装饰、自然流动感
- **毛玻璃效果**：侧边栏和顶栏使用 `backdrop-filter: blur()`
- **柔和色彩阴影**：蓝色/黄色色调的阴影，避免纯黑阴影

### 1.2 品牌色来源

从 Logo 提取三个品牌色：

| 色彩 | 来源 | 色值 |
|------|------|------|
| 宝蓝（主色） | 工作服 | `#4A7FB5` |
| 金黄（辅色） | 安全帽 | `#E8B84B` |
| 珊瑚红（危险色） | 披风 | `#D4726A` |

```css
:root {
  /* 基础色 */

  /* 品牌色 */
  --primary: #4A7FB5;         /* 主色 - 宝蓝 */
  --primary-fg: #FFFFFF;      /* 主色上的文字 */
  --secondary: #E8B84B;       /* 辅色 - 金黄 */
  --secondary-fg: #FFFFFF;    /* 辅色上的文字 */
  --destructive: #D4726A;     /* 危险色 - 珊瑚红 */

  /* 中性色 */
  --accent: #E6DCCD;          /* 强调背景 - 暖灰 */
  --accent-fg: #4A4A40;       /* 强调文字 */

  --muted-fg: #78786C;        /* 弱化文字 */
  --border: #DED8CF;          /* 边框色 */

```

### 2.2 阴影系统

```css
:root {
  --shadow-soft: 0 4px 20px -2px rgba(74, 127, 181, 0.15);    /* 常规卡片阴影 */
  --shadow-float: 0 10px 40px -10px rgba(232, 184, 75, 0.2);  /* 浮动元素阴影 */
  --shadow-hover: 0 20px 40px -10px rgba(74, 127, 181, 0.15); /* 悬停阴影 */
}
```

> **注意**：所有阴影都带有品牌色调（蓝色或黄色），不使用纯黑阴影。

### 2.3 字体系统

```css
:root {
```

| 用途 | 字体 | 字重 | 示例场景 |

| 正文/按钮 | Nunito (无衬线) | 400-600 | 表格内容、按钮文字、表单标签 |

### 2.5 圆角系统

| 类型 | 圆角值 | 使用场景 |
|------|--------|----------|
| Pill（胶囊） | `999px` | 按钮、输入框、标签、分页、状态badge |
| 卡片（不对称） | `24px 16px 24px 16px` 等 | 统计卡片、信息卡片（交替使用不同组合） |
| 容器（大圆角） | `24px` | 表格容器、筛选工具栏 |
| 导航项 | `12px` | 侧边栏菜单项 |
| 小元素 | `8px` | 操作按钮hover背景、小交互元素 |

**不对称圆角组合**（4个统计卡片交替使用）：
```css
/* 卡片1 */ border-radius: 24px 16px 24px 16px;
/* 卡片2 */ border-radius: 16px 24px 16px 24px;
/* 卡片3 */ border-radius: 24px 24px 16px 16px;
/* 卡片4 */ border-radius: 16px 16px 24px 24px;
```

### 2.6 间距系统

| 场景 | 间距值 |
|------|--------|
| 页面内容区 padding | `24px 28px 20px` |
| 卡片内部 padding | `20px` |
| 顶栏/侧边栏 padding | `0 28px` / `0 20px` |

| 元素间距（小） | `12px`（gap） |

## 三、全局效果

### 3.1 纸质纹理（必须）

所有页面必须覆盖纸张噪点纹理：

```css
body::before {
  content: '';
  position: fixed;
  top: 0; left: 0; right: 0; bottom: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)' opacity='0.04'/%3E%3C/svg%3E");
  background-repeat: repeat;
  pointer-events: none;
  z-index: 9999;
  mix-blend-mode: multiply;
  opacity: 0.5;
}
```

### 3.2 背景 Blob 装饰

主内容区右上角和左下角各一个模糊色块：

```css
.main-content::before {
  content: '';
  position: absolute;
  top: -60px; right: -40px;
  width: 200px; height: 200px;
  background: var(--primary);
  border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%;
  filter: blur(80px);
  opacity: 0.06;
  pointer-events: none;
  z-index: 0;
}
.main-content::after {
  content: '';
  position: absolute;
  bottom: -40px; left: -30px;
  width: 160px; height: 160px;
  background: var(--secondary);
  border-radius: 40% 60% 70% 30% / 40% 70% 30% 60%;
  filter: blur(60px);
  opacity: 0.05;
  pointer-events: none;
  z-index: 0;
}
```

### 3.3 毛玻璃效果

侧边栏和顶栏使用毛玻璃：

```css
/* 侧边栏 */
.sidebar {
  background: rgba(253,252,248,0.85);
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  border-right: 1px solid rgba(222,216,207,0.5);
}

/* 顶栏 */
.topbar {
  background: rgba(253,252,248,0.7);
  backdrop-filter: blur(8px);
  -webkit-backdrop-filter: blur(8px);
  border-bottom: 1px solid rgba(222,216,207,0.3);
}
```

## 八、交互规范

### 8.1 动效原则

- 所有过渡使用 `transition: all 0.3s ease`
- 悬停效果：卡片 `translateY(-2px)`，按钮 `scale(1.05)`
- 按下效果：按钮 `scale(0.95)`
- 禁止使用弹跳、闪烁等突兀动效

### 8.2 超期提醒

| 场景 | 规则 | 样式 |
|------|------|------|
| 工单超期 | 创建后 24h 未派单 | 行背景 `rgba(212,114,106,0.06)` |
| 回访超期 | 完成后 3 天未回访 | 行背景 `rgba(212,114,106,0.06)` + 标红 |
| 投诉超期 | 超过 7 天未处理 | 行背景 `rgba(212,114,106,0.06)` + 标红 |

### 8.3 空状态

列表无数据时显示空状态插图 + 提示文字，使用 `var(--muted-fg)` 颜色。

- 侧边栏可收起为图标模式（64px）

- 表格改为横向滚动
