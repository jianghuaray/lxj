# 2026-05-05 工作记录

## 手机号验证移除 (VolunteerAdd)

### 修改内容
- **前端 VolunteerAdd.vue**：
  - 输入框改为只允许输入数字：`@input="form.phone = form.phone.replace(/[^0-9]/g, '')"`
  - 输入类型改为 `type="text"` + `inputmode="numeric"`（移动端更好用）
  - 删除正则验证 `/^1[3-9]\d{9}$/`，只保留必填校验
- **后端 volunteers.js**：
  - 删除 `const { validatePhone } = require('../utils/sanitize')` import
  - 删除 `validatePhone(phone)` 调用，不再校验手机号格式

### 效果
- 手机号输入框只能输入数字，不能输入字母或符号
- 不再强制要求 11 位手机号格式，只要填了数字即可提交
