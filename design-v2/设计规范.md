### 1.2 工单状态流转（🔴 重大变更）

```
旧流程：待派单 → 已派单 → 施工中 → 已完成 → 已回访
                              ↑
                           已删除

新流程：待派单 → 已派单 → 已完成 → 已回访
                    ↘ 已取消
                    ↘ 咨询单
```

> **业务背景**：师傅联系客户和上门维修的过程对接线员是黑盒，接线员不需要知道中间进展，只需要在师傅反馈完成后录入结果。已派单工单不需要超时提醒。

**状态 Badge 颜色（去掉"施工中"）：**

| 状态 | 类名 | 背景色 | 文字色 |
|------|------|--------|--------|
| 待派单 | pending-dispatch | `rgba(74,127,181,0.12)` | `#4A7FB5` |
| 已派单 | dispatched | `rgba(232,184,75,0.12)` | `#B8922E` |
| ~~施工中~~ | ~~in-progress~~ | ~~已删除~~ | ~~已删除~~ |
| 已完成 | completed | `rgba(74,127,181,0.15)` | `#3D6A9A` |
| 已回访 | followed-up | `rgba(74,127,181,0.1)` | `#4A7FB5` |
| 已取消 | cancelled | `rgba(212,114,106,0.1)` | `#D4726A` |
| 咨询单 | consultation | `rgba(120,120,108,0.1)` | `#78786C` |

## 二、设计系统概述

### 1.1 设计理念

采用 **Organic 自然风格**，核心特征：

- **侘寂美学（Wabi-sabi）**：不完美中的美感，柔和、温暖、自然
- **纸质纹理**：全局覆盖纸张噪点纹理，营造手工质感
- **有机形态**：不对称圆角、blob 装饰、自然流动感
- **毛玻璃效果**：侧边栏和顶栏使用 `backdrop-filter: blur()`
- **柔和色彩阴影**：蓝色/黄色色调的阴影，避免纯黑阴影

### 1.2 品牌色来源

从 Logo 提取三个品牌色：

| 色彩 | 来源 | 色值 |
|------|------|------|
| 宝蓝（主色） | 工作服 | `#4A7FB5` |
| 金黄（辅色） | 安全帽 | `#E8B84B` |
| 珊瑚红（危险色） | 披风 | `#D4726A` |

## 三、全局效果

### 3.1 纸质纹理（必须）

所有页面必须覆盖纸张噪点纹理：

```css
body::before {
  content: '';
  position: fixed;
  top: 0; left: 0; right: 0; bottom: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)' opacity='0.04'/%3E%3C/svg%3E");
  background-repeat: repeat;
  pointer-events: none;
  z-index: 9999;
  mix-blend-mode: multiply;
  opacity: 0.5;
}
```

### 3.2 背景 Blob 装饰

主内容区右上角和左下角各一个模糊色块：

```css

.main-content::after {
```

## 八、交互规范

### 8.1 动效原则

- 所有过渡使用 `transition: all 0.3s ease`
- 悬停效果：卡片 `translateY(-2px)`，按钮 `scale(1.05)`
- 按下效果：按钮 `scale(0.95)`
- 禁止使用弹跳、闪烁等突兀动效

### 8.2 超期提醒

| 场景 | 规则 | 样式 |
|------|------|------|
| 工单超期 | 创建后 24h 未派单 | 行背景 `rgba(212,114,106,0.06)` |
| 回访超期 | 完成后 3 天未回访 | 行背景 `rgba(212,114,106,0.06)` + 标红 |
| 投诉超期 | 超过 7 天未处理 | 行背景 `rgba(212,114,106,0.06)` + 标红 |

### 8.3 空状态

列表无数据时显示空状态插图 + 提示文字，使用 `var(--muted-fg)` 颜色。
